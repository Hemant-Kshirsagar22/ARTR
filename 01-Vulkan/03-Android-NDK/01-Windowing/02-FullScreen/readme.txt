steps for creating template
1. remove test dirs from ./src/main.
2. remove java dir from ./src/main.
3. update app_name inside res/values/string.xml
4. paste your icon files inside res/mipmap* in all folders starting with mipmap
5. Update the AndroidManifest.xml file.
6. Update namespace name for every new project from app/build.gradlep
7. Update rootProject.name from settings.gradle file
